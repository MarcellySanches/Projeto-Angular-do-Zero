
# Projeto Angular do Zero

## Sobre o projeto 
Esta demanda foi realizada durante o bootcamp de Web FullStack da SoulCode. <br>
O intuito era criar um produto mínimo viável para um cenário de curso ou tutoriais sobre Angular.
